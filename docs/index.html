Workshop on Image Processing using OpenCV

The workshop is designed to introduce students with the basic concepts in image processing and how those techniques can be applied using library like OpenCV.

Aim:

  • To introduce the basic concepts of Image Processing
  • To attain familiarity with the OpenCV library
  • To grasp the fundamental concepts behind Image Segmentation

Highlights:

  • A brief of OpenCV and Python
  • Introduction to Digital Image Processing
  • Introduction to pixels and pixel Manipulation
  • Image types and different color channels
  • Providing an idea about basic image operations
  • Providing the interaction of custom interfaces to work with images
  • An overview of Image Segmentation
  • An overview of Object Detection

What we'll cover:

  • Image Processing Libraries
  • OpenCV
  • Digital Image
  • Pixels
  • Neighbourhood of pixels
  • Color channels
  • Filtering and Scaling of Images
  • RGB, BGR and Binary Images
  • Segmentation
  • Thresholding
  • Edge Detection
  • Feature Detection
